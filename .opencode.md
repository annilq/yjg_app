# 云建管 Flutter 项目上下文

## 项目信息
- 名称：云建管（建筑企业 SaaS App）
- 路径：~/Documents/work/jg/flutter_app
- 架构：Feature-First + MVVM

## 技术栈
- Flutter 3.41.2 / Dart 3.11.0
- 状态管理：flutter_riverpod ^2.5.0
- 导航：go_router ^14.0.0
- 网络：dio ^5.5.0
- 代码生成：freezed + json_serializable

## 目录规范
```
lib/features/{模块}/
├── providers/          # Riverpod 状态管理
├── models/             # 数据模型（freezed）
├── presentation/
│   ├── screens/        # 页面
│   └── widgets/        # 组件
└── service/            # 数据服务
```

## 现有模块
- auth：认证（登录、注册、密码重置）
- home：首页
- contact：通讯录
- notices：通知
- office：办公
- profile：个人中心
- webview：WebView 嵌套
- workflow：工作流程
- settings：设置（主题、语言、通知、关于）

## API 规范
- 基础 URL：http://api.jianguanoa.com/1.0/
- 请求方式：POST
- 认证：时间戳 + 签名

## 开发规则
1. 优先使用 AppBarComponent 作为导航栏
2. 状态管理统一使用 Riverpod StateNotifierProvider
3. UI 组件优先使用 shared/widgets 中的公共组件
4. 路由统一在 routes/app_router.dart 中配置
5. 运行命令：flutter run -d macos
